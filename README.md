💰 Expense Tracker App

👋 A simple and powerful Flutter Expense Tracker Application to manage daily expenses and track spending habits efficiently.

🚀 Overview

A clean and responsive mobile application built using Flutter that helps users add, view, and manage expenses with an easy-to-use interface and smooth performance.

✨ Features
➕ Add daily expenses
🗑️ Delete expenses
📊 View total spending summary
📅 Date-wise tracking
📱 Clean UI/UX design
⚡ Fast & lightweight performance
📴 Offline support
🧠 About the Project

This project was built to improve my Flutter development skills and understand real-world app architecture including state management, UI design, and local storage handling.
